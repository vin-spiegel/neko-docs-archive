## MakeSturn(Single)
유닛에게 스턴(Stun)을 적용시킵니다.

#### 선언
```cs
public void MakeSturn(float time)
```

#### 매개 변수(인자)

|타입|이름|설명|
|:-|:-|:-|
|System.Single|time|적용시간|

#### 예제: 유닛을 3초간 스턴시키기
```lua
unit.MakeSturn(3)
```
