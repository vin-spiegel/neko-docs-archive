## SetInitPos(Single, Single)
몬스터의 최초 생성 위치를 변경한다

#### 선언
```cs
public void SetInitPos(float posX, float posY)
```

#### 매개 변수(인자)

|타입|이름|설명|
|:-|:-|:-|
|System.Single|posX|생성할 위치 X|
|System.Single|posY|생성할 위치 Y|
